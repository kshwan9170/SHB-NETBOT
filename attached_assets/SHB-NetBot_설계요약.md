
# SHB-NetBot - 신한은행 내부 네트워크 챗봇

## ✅ 프로젝트 개요
- 신한은행 직원들이 자주 묻는 네트워크 관련 질문(예: 스윙 접속, IP 확인 등)을 자동으로 응답하는 웹 기반 챗봇입니다.
- 해커톤에서 MVP 형태로 구현하되, 향후 사내 메신저 연동을 통해 실제 업무 환경에 도입하는 것을 최종 목표로 합니다.

---

## ✅ 기술 스택 및 구조

### 핵심 구성
| 구성 요소 | 사용 기술 |
|------------|------------|
| 웹 인터페이스 | Streamlit |
| 챗봇 엔진 | OpenAI GPT-3.5 (ChatCompletion API) |
| API Key 관리 | `.env` 파일 (환경변수) |
| 문서 검색 (RAG) | Chroma (벡터 DB) |
| 문서 유형 | PDF, PPTX, DOCX, TXT 등 |
| 로그 저장 (선택) | SQLite 또는 JSON |

### 동작 흐름 (RAG 구조 기준)
1. 내부 문서에서 텍스트 추출
2. 문장을 벡터(숫자)로 변환하여 Chroma DB에 저장
3. 사용자가 질문하면 해당 질문도 벡터화
4. 가장 유사한 문장을 찾아 GPT에게 전달
5. GPT가 전문가처럼 자연어로 응답 생성

---

## ✅ 개발 및 실행 순서

### 1단계: 초기 세팅
- Replit 프로젝트 생성 (Python, 비공개)
- `.env` 파일에 `OPENAI_API_KEY` 저장
- `requirements.txt`에 패키지 추가:
  ```
  openai
  streamlit
  chromadb
  ```

### 2단계: 기본 챗봇 기능 구현
- Streamlit으로 입력창 + 응답 출력 UI 생성
- GPT-3.5 API 호출 테스트 (model="gpt-3.5-turbo")

### 3단계: 문서 기반 기능 추가
- PDF, PPT 등에서 텍스트 추출
- 문장 단위로 분리 후 Chroma에 임베딩 저장
- 질문과 유사한 문서 내용 검색하여 응답에 활용

### 4단계: 시연 및 마무리
- 질문 리스트 구성
- 챗봇 작동 영상 녹화 (Loom, OBS 등)
- 발표자료 제작
